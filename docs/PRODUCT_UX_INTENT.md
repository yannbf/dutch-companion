# Product & UX Intent

Concise intent guide for designers and AI contributors.

## Product Purpose

Dutch Companion is a focused Dutch-language learning app that helps learners practice through short, repeatable exercises.

Core value:

- practice frequently
- get immediate feedback
- build confidence across grammar, vocabulary, sentence construction, and pronunciation

## Primary User

A learner of Dutch (likely beginner to intermediate) who wants practical repetition rather than long lessons.

Typical context:

- short mobile sessions
- self-study
- daily repetition and spaced review habits

## UX Principles

1. **Fast to start**
   - Each exercise should be playable in a few taps.

2. **Small learning loops**
   - Setup → interaction → immediate feedback → summary → replay.

3. **Low cognitive friction**
   - Clear labels, simple controls, obvious primary action.

4. **Consistent exercise patterns**
   - Similar setup and result patterns across exercise types.

5. **Progress continuity**
   - User settings/favorites/progress persist between sessions.

## Exercise Intent by Feature

- **Verb tenses**
  - Drill irregular verb forms quickly (recognition + recall).

- **Vocabulary flashcards**
  - Build word familiarity with optional translation/audio support.

- **Vocabulary match**
  - Strengthen Dutch↔English pairing speed.

- **De/Het**
  - Reinforce article selection for nouns.

- **Separable verbs / om...te sentence builder**
  - Practice sentence structure and word order.

- **Wheel of Fortune (sentence generator)**
  - Encourage productive sentence practice by combining pronoun/tense/verb prompts.

- **Pronunciation practice**
  - Expose learners to authentic pronunciation in context.

- **Vocabulary browser + stats**
  - Support review and self-monitoring outside active drills.

## UX Use Cases

1. **Quick daily drill (2–5 min)**
   - Open app → Exercises → pick recent drill → complete short session.

2. **Targeted weakness practice**
   - Configure chapter/category/difficulty → run focused rounds.

3. **Audio-supported learning**
   - Enable voice mode and replay words while studying.

4. **Progressive expansion**
   - Start with selected levels/chapters, broaden over time.

## Non-Goals (for now)

- Not a full curriculum/LMS.
- Not teacher-managed classroom software.
- Not long-form course content.

## Change Management Rule

If a PR changes exercise intent, user flow, or learning outcomes, update this file in the same PR.
